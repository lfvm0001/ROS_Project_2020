This txt file contains all the step to install and run the turtlebot example.
The steps were taken from: https://automaticaddison.com/how-to-launch-the-turtlebot3-simulation-with-ros/

1. Install dependent packages
Create the next directory where you want to have the turtlebot example:

    mkdir /catkin_turtlebot3
    cd /catkin_turtlebot3/
    mkdir src
    cd src/
    git clone https://github.com/ROBOTIS-GIT/turtlebot3_msgs.git
    git clone https://github.com/ROBOTIS-GIT/turtlebot3.git
    cd ../ && catkin_make

2. Adding turtlebot3 model to the bashrc file

    open "~/.bashrc" file and add the following line at the end of the file: export TURTLEBOT3_MODEL=burger

    source ~/.bashrc

3. Download turtlebot simulation files

    cd /catkin_turtlebot3/src/
    git clone https://github.com/ROBOTIS-GIT/turtlebot3_simulations.git
    cd ../ && catkin_make

4. Simulation of turtlebot

    source devel/setup.bash
    roslaunch turtlebot3_fake turtlebot3_fake.launch

    Note: If you want to move the turtlebot, open a new terminal, source the devel/setup.bash file and run:

    roslaunch turtlebot3_teleop turtlebot3_teleop_key.launch
